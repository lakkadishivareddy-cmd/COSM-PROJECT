# Heights and Weights Statistical Analysis

This project presents a complete statistical and machine learning analysis of the Heights and Weights dataset using Python. It includes data preprocessing, visualization, linear regression modelling, and essential hypothesis testing techniques. The implementation is optimized for Google Colab, ensuring simplicity, readability, and ease of execution for academic and research use.

This study investigates the relationship between height and weight using statistical analysis and predictive modelling. Key tasks include:

- Implementing Linear Regression  
- Performing statistical tests:
  - One-sample t-test  
  - Independent two-sample t-test
